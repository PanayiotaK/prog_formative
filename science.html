Computer Science in Science

Computer science can be used to help model complicated systems, such as the Earth's atmosphere or the universe, which allows scientists to demonstrate and quickly test their hypotheses. Computers can process a large amount of data, which is useful for scientific research requiring lots of computation on a dataset. The field of data science uses scientific processes to extract information from data using methods including machine learning; this data can be used in scientific research.

Key examples

  • Rockets
    • Computer science has been used to develop rockets for space exploration and research
    • Errors in code can be catastrophic, for example the Ariane 5 disaster
